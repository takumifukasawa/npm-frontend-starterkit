# npm-frontend-starterkit


```
使い方

$ npm i

$ npm start
  -> pug,scss,watchifyのwatchと簡易サーバーが立ち上げる

$ npm run build
  -> コンパイルだけさせたいとき

$ npm run sprity-front-pc
$ npm run sprity-front-sp
  -> spはretina用のスプライト画像が書き出される

```
